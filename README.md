# Portfolio Website

A modern personal portfolio built with React, Vite, and Tailwind CSS.

## Highlights
- Project gallery with category and year filters (sorted newest first)
- Motion-driven sections using Framer Motion
- Light/dark theme styling
- Contact form powered by EmailJS with basic rate limiting
- Fully responsive layout

## Tech Stack
- React 19
- Vite 6
- Tailwind CSS 4 + DaisyUI
- Framer Motion
- EmailJS

## Project Data
Projects live in `src/assets/data.json`.

## Getting Started
```bash
npm install
npm run dev
```

## Build And Preview
```bash
npm run build
npm run preview
```

## Scripts
- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run lint`

## EmailJS Setup
Update the EmailJS identifiers in `src/pages/Contact.jsx`:
- Service ID
- Template ID
- Public key
